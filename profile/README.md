## What is FanControl?

FanControl is a powerful Windows hardware cooling utility that gives users complete control over CPU, GPU, case, and liquid cooling fan behavior. It enables fully customizable fan curves based on multiple temperature sensors, helping maintain optimal cooling performance while reducing unnecessary system noise during everyday workloads and demanding applications.
